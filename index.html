Tentu website ini gratis!!!. Semua item yang ada dalam website ini dapat diakses secara gratis tanpa biaya sepeserpun, hanya saja pengguna harus online saat mengakses website "ProjectTiga".

User tidak perlu mendaftar. Pengembangan website "ProjectTiga" ditujukan untuk peserta didik maka dari itu tidak ada tahapan pendaftaran.

Website ini aman untuk digunakan. Sejauh pengembang melakukan riset terhadap website, pengembang memastikan bahwa data dari user tidak akan disimpan dan dikelola.

User dapat menghubungi pengembang. Apabila terjadi error pada saat mengakses website, user dapat menghubungi pengembang melalui sosial media yang telah disediakan.

Tentu tersedia. Sejak awal pengembang telah melakukan riset terhadap website guna memudahkan user untuk mengakses website yang disediakan.
